brom
www.brom.horse

brom is a highly configurable CLI for recording HTTP transactions and improving security practices, designed for use in local environments and CI tools. In its interactive mode, brom proxies your server, recording HTTP transactions as they stream through, and injecting a script into HTML documents to record third-party AJAX calls. When run in its automated mode, brom traverses Express application structures to identify all registered routes and their supported methods, sending dumb requests to each one. brom provides a rule syntax to allow developers to create their own rules, effectively turning brom into an extensible test suite. Audit results are displayed in a React/Redux GUI that is served locally as part of the npm package. Each result encompasses a single HTTP transaction, displaying the method, route, request, response, and all headers, as well as warnings and metadata.

Walker
github

Walker is a personal security application to keep an eye on you when you walk from one destination to another. Current personal security applications require you to keep your phone in your hand at all times (Noonlight/SafeTrek) or to use proprietary in-app interfaces. Holding your phone out in the open can make you a more enticing target, and relying on unfamiliar interfaces in high-stress situations can be confusing. Walker alleviates this by interfacing with the user via SMS.
